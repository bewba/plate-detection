# Licence-Plate-Detection-and-Recognition-using-YOLO-V8-EasyOCR

This project performs automatic license plate detection and recognition using:

YOLOv8 for detecting license plates in images/video

EasyOCR for extracting text from detected plates

Build and start the application: `docker compose up -d --build`

This dockerfile is especially made to use ROCm on my AMD RX 6600XT
